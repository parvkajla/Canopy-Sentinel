# Canopy Sentinel

An Android application for tracking and monitoring deforestation using satellite data and real-time alerts.

## Features

- Real-time deforestation monitoring
- Interactive map with satellite imagery
- Deforestation alerts and notifications
- Area statistics and analytics
- Location-based search
- Material Design 3 UI

## Requirements

- Android Studio Arctic Fox or later
- Android SDK 26 or higher
- Google Maps API key
- Gradle 7.0 or higher

## Setup

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/CanopySentinalG.git
```

2. Open the project in Android Studio

3. Add your Google Maps API key:
   - Open `app/src/main/res/values/strings.xml`
   - Replace `YOUR_MAPS_API_KEY_HERE` with your actual API key

4. Build and run the project

## Architecture

- MVVM Architecture
- Material Design Components
- Google Maps SDK
- AndroidX libraries
